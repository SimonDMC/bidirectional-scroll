# Bi-directional Scroll Demo

Demo: https://simondmc.com/bidirectional-scroll-demo

This project is a showcase/demo of an infinitely scrollable list of items in both directions.

I initially started working on it for the purposes of being able to swipe to switch months on
a calendar in my personal journal app. However, it ended up being a more complicated feature
than I thought it would be, so I published it as its own separate project, in case it could be
useful as a starting point for anyone attempting to make a similar component.

The source code is published under the MIT license.
